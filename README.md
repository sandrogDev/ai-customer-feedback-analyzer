# AI-Powered Customer Feedback Analyzer 📊🤖

An automated data pipeline written in Python that ingests raw, unstructured customer reviews, analyzes them using Google Gemini AI (`gemini-2.5-flash`), and exports structured insights into a production-ready CSV report.

This tool is designed for e-commerce owners and managers who receive hundreds of text reviews daily and need instant, actionable data to improve their product, shipping, or customer support without reading every single line manually.

---

## 🛠️ Key Features
- **Automated Ingestion:** Reads disoriented, raw text reviews directly from a source file (`reviews.txt`).
- **Advanced Sentiment Analysis:** Categorizes every review into strict categories: *Positive, Negative, or Neutral*.
- **Critical Area Tagging:** Automatically identifies which company department is involved (*Product, Shipping, Support, Website*).
- **Chirurgic Summarization:** Uses strict Prompt Engineering to force the AI to summarize the core issue in 4 words or less.
- **Robust Pipeline:** Built-in rate-limiting handling (`time.sleep`) to prevent server overloads and API crashes.
- **Security First:** Protects private API keys securely using environment variables (`python-dotenv`).

---

## 📂 Project Structure
- `analyzer.py` - The main Python script containing the pipeline logic.
- `reviews.txt` - Input file containing sample raw user feedback.
- `results.csv` - The final structured spreadsheet output generated by the AI.
- `.env.example` - A template file showing how to configure your environment keys securely.

---

## 🚀 How It Works (The Prompt Strategy)
The core value of this script relies on a strict enterprise-level System Prompt that forces the AI to reply **only** with structured tokens separated by a pipe (`|`) character, ignoring conversational fluff. 
Python then grabs this raw output, splits it, and builds a clean table ready for Excel or Google Sheets.

## 🇮🇹 VERSIONE ITALIANA (Presentazione Progetto)

### Pipeline di Analisi Automatica dei Feedback Clienti con Intelligenza Artificiale
Questo software automatizza un processo aziendale noioso e dispendioso in termini di tempo: la lettura e la categorizzazione di centinaia di recensioni o email di supporto clienti.

### Perché è utile per un'azienda?
Un manager o il proprietario di un e-commerce non ha il tempo di leggere 1.000 recensioni al mese. Questo script fa il lavoro di una settimana in meno di 2 minuti. Invece di testi disordinati, il manager otterrà un file Excel ordinato che dice subito, ad esempio: *"Questo mese il 70% dei voti negativi riguarda i ritardi del corriere"*. Questo permette di prendere decisioni di business immediate e mirate.

### Competenze Tecniche Dimostrate in questo Progetto:
- **Ingegneria dei Dati:** Lettura, pulizia (rimozione spazi vuoti) e scrittura di file esterni (CSV/TXT).
- **Integrazione AI di livello Professionale:** Connessione sicura tramite API con l'SDK ufficiale di Google Gemini.
- **Prompt Engineering Avanzato:** Controllo rigido dell'output dell'AI per scartare i testi inutili e ottenere solo dati strutturati.
- **Sicurezza Aziendale:** Gestione delle credenziali e delle password tramite variabili d'ambiente nascoste (`.env`).
- **Ottimizzazione di Rete:** Gestione del *Rate Limiting* dei server tramite pause controllate per evitare blocchi temporanei.

## 🔧 Installation & Setup

1. Clone this repository or download the files.
2. Activate your Virtual Environment:
```bash
   .\venv\Scripts\activate
   ```
3. Install the required dependencies:
```bash
   pip install google-genai python-dotenv
   ```
4. Create a `.env` file in the root directory and add your secret Gemini API Key:
```text
   GEMINI_API_KEY=your_actual_api_key_here
   ```
5. Run the analyzer:
```bash
   python analyzer.py
   ```
